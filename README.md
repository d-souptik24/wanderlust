## 🌟 What is Wanderlust?

**Wanderlust** is a passion project and a real-world deep dive into **full-stack web development**. It's an Airbnb-style rental marketplace where travellers can discover beautiful properties, and hosts can list their spaces for the world to see.

Built entirely from scratch using the **MERN stack**, every single feature — from the database to the browser — has been hand-crafted to understand how real-world web applications are built and scaled.

> 💡 *Not just a clone — a learning-first, production-minded web application.*

## 🚀 What We've Built So Far

### 🗄️ Real Persistent Database

> No dummy arrays. No in-memory storage. This is the real deal.

The app is wired to a live **MongoDB** database named `wanderlust`. Every listing you add lives in the database and survives server restarts — just like a real production app.

---

### 🔁 Complete Listing Lifecycle (CRUD)

From creation to deletion, we handle the **full lifecycle of a property**:

```
➕ Create  →  📖 Read  →  ✏️ Update  →  🗑️ Delete
```

Users can walk through the entire journey of managing a rental property right from the browser — no backend knowledge needed.

---

### 🎨 Polished, Responsive UI

No raw HTML wireframes here. The UI is built to feel like a real product:

- 🖼️ **Card grid layout** for the listing catalog — clean, scannable, beautiful
- 📄 **Detail view** to spotlight each property with its image, price, and location
- 📝 **Smooth forms** for adding and editing listings
- 🔝 **Global navbar & footer** across every single page
- 🅰️ Custom styling using the *Plus Jakarta Sans* font for a modern, refined look

---

### 🌱 Seed Data — Realistic From Day One

The project comes loaded with a **curated set of sample listings** from real-looking places around the globe. One command resets and repopulates the entire database — perfect for testing and development.

---

### 🏗️ Clean & Scalable Architecture

Everything is organized with growth in mind — following the battle-tested **MVC pattern**:

```
📁 Airbnb Clone/
 ├── 🟢 app.js              → Server, routes & all middleware
 ├── 📁 models/
 │    └── listing.js        → MongoDB schema for listings
 ├── 📁 views/              → All EJS page templates
 ├── 📁 public/
 │    └── css/style.css     → Custom design system
 ├── 📁 init/
 │    ├── index.js          → Database seeder script
 │    └── data.js           → Sample listings data
 └── 📁 docs/               → Project documentation
```
